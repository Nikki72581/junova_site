# Junova — Next.js App Router Starter

## Commands
```bash
npm install
npm run dev
```

## What’s inside
- Next.js 14 + TypeScript + Tailwind
- Global brand tokens at `app/styles/junova.tokens.css`
- Wordmark and favicons in `/public`
- Clean homepage at `/`
- Sanity route at `/_debug` to verify tokens
